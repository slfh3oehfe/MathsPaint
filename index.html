How to Use

Getting Started

1

Download

Click the Download button on the home page. You'll get a single zip file containing everything you need.

2

Extract

Right click the zip and choose Extract All. Keep the folder structure intact — the icons folder must stay alongside the HTML file.

3

Enter your licence key

Open the app and enter your lifetime licence key. You only need to do this once — it saves automatically.

4

Set up your profile

Choose a display name and optional profile picture. You can change these anytime in Settings.

5

Start creating

You're all set. Jump into the Paint tab to draw or the Graph tab to start plotting.

Paint Tab

The paint canvas fills your entire window. Switch tools from the toolbar at the top.

Multiple canvases

Open multiple canvases with the + button in the tab bar. Switch between them without losing your work.

Colour palette

Click the palette icon next to the colour circle to open your saved swatches. Right-click a swatch to remove it.

Select tool

Draw a selection, then drag to move it. Press Ctrl+C to copy, Delete to remove, Escape to commit.

Graph Tab

A 700×700 XY coordinate graph with zoom, pan and function plotting.

Function plotter

Type any expression like x^2 or sin(x) into the f(x) bar and press Plot to draw the curve.

Point table

Click the table icon in the graph toolbar to view, edit and add plotted points as a spreadsheet.

Snap to grid

Toggle snap mode to lock new points to the nearest grid intersection.

Documentation

Overview

PaintX is a desktop application built with C# WinForms and WebView2. It is a creative workspace combining a full paint suite and a mathematical graph plotter.

Features

Paint Suite

Draw, erase, line, circle, rect, select. Multiple canvases with undo/redo.

Graph Plotter

700×700 XY grid. Plot points, lines, labels, free draw and function curves.

Themes

8 built-in colour themes plus custom accent colour. Themes update the entire UI instantly.

Bilingual

Full English and Russian language support. Switch at any time from Settings.

Backgrounds

Set any image or GIF as the app background with adjustable opacity.

Function Plotter

Plot sin(x), cos(x), x^2, sqrt(x) and any custom expression.

Notifications

Toast notifications for save, copy and graph actions.

Colour Palette

Save favourite colours as swatches for instant access while painting.

Keyboard Shortcuts

Ctrl + C

Copy canvas or copy selection (when select tool is active).

Ctrl + V

Paste image from clipboard onto the canvas.

Delete / Backspace

Delete active selection.

Escape

Commit and deselect current selection.

Enter (in f(x) bar)

Plot the typed function on the graph.

FAQ

Is PaintX free?

Yes. A lifetime licence key gives you permanent access to all features with no subscription.

What are the system requirements?

Windows 10 or later. WebView2 Runtime is required — most modern Windows installs already have it.

Where is my data stored?

All settings, profile and session data are stored locally. Nothing is sent externally.

Can I use my own background image?

Yes. Go to Settings → Appearance → App Background and paste an image URL or choose a file from your device.

Get a Key

Your Licence Key

PaintX uses a one-time lifetime licence key. You enter it once on first launch and it saves permanently — no logins, no subscriptions, no expiry.

How to get a key

Keys are distributed manually by the admin. To get your licence key, contact the admin directly and they will generate one for you.

Why do we use a licence?

We use a simple one-time key system for a few reasons:

Control distribution

The key lets us control who has access to the app without needing accounts, servers or a backend. One key, one download, done.

No accounts needed

There is no server storing your data. The key is verified locally — your name, settings, canvases and profile never leave your machine.

Prevent mass redistribution

Without a key system anyone could share the app freely and we would have no way to manage updates or support. The key keeps the user base intentional.

It is lifetime — not a subscription

You pay once (or receive the key) and it never expires. There are no renewals, no monthly fees and no features locked behind tiers.

How to use your key

1

Open PaintX

Launch the app for the first time. You will be greeted with the setup wizard.

2

Enter your key

Type or paste your generated key (starting with PX-) into the licence key field.

3

Continue setup

Set your display name and profile picture, then click Start. Your key is saved — you will never need to enter it again.
